# C
- It's representative of what we're going to generally call source **code**.
- **Machine code** is what the computers understand.
- ~={yellow}**Compiler**=~ takes **source code** as input, produces **machine code** as output.
## GUI and CLI
## VScode
- `$ code hello.c`
- `$ make hello`
- `./hello`
	- What the *dot slash* means is that after having just made a program called hello, that program's going to end up in my current folder.
## Header Files
- A **library**(库) is code someone else wrote that you can use.
